# 公众号文章半自动工作流（已验证）

## 核心结论

| 编辑器 | 粘贴保留样式 | 结论 |
|--------|-------------|------|
| 秀米 | ❌ 全部剥离 | 不适用 |
| 135编辑器 | ✅ 完美保留 | **使用135** |

## 完整流程（6步）

### 第1步：写作
在 Claude Code 对话中，用户给出选题 → AI 写文章 → 保存为 `article-enhanced.md`

**Markdown 增强标记**（wechat-publisher 支持）：
- `**文本**` → 加粗（黄绿渐变高亮）
- `==文本==` → 黄色荧光笔
- `++文本++` → 蓝色高亮
- `&&文本&&` → 绿色高亮
- `!!文本!!` → 红色强调

**硬性规范**：
- 字数：850-1050字（正文）
- 结构：4-6小节，单段≤4行
- 三类内容：养生健康 / 中年家庭情感 / 生活实用技巧
- 禁用：网络热梗、标题党、偏方谣言

### 第2步：AI生图（Gemini）
Playwright 打开 gemini.google.com → 输入4个配图提示词 → 下载 → 重命名为 `配图一~四.png`

**配图提示词模板**：
```
写实摄影风格：{场景描述}，光线自然柔和，生活化中式家庭场景，高清竖构图
```

### 第3步：排版渲染
```bash
cd C:\Users\24763\.claude\skills\wechat-publisher\scripts
python generate_daily.py \
  --topic "选题" \
  --type "养生健康|中年家庭情感|生活实用技巧" \
  --input article-enhanced.md \
  --images 图片目录 \
  --output 输出目录
```

**主题自动匹配**：
| 话题类型 | 主题 | 风格 |
|----------|------|------|
| 养生健康 | sage-premium | 鼠尾草墨绿+象牙白 |
| 家庭情感 | warm-editorial | 栗色+米白+衬线体 |
| 实用技巧 | refined-blue | 专业蓝+白 |

### 第4步：生成最终HTML
脚本输出两个版本：
- `公众号文章-日期-内嵌版.html` (11MB) → 配图base64内嵌，**135粘贴用这个**
- `公众号文章-日期-秀米粘贴版.html` (11KB) → 轻量占位版，存档

### 第5步：粘贴到135编辑器
1. 打开135编辑器：https://www.135editor.com/beautify_editor.html
2. 新建文章
3. 浏览器打开「内嵌版.html」→ Ctrl+A → Ctrl+C
4. 点入135编辑区 → Ctrl+V
5. 检查格式 → 点「保存同步」→ 「复制使用」

### 第6步：发布
135编辑器 → 「同步公众号」或「复制使用」→ 粘贴到微信公众号后台 → 发布

## 关键教训

1. **秀米不可用** — 它强制剥离所有CSS样式，只能纯文本
2. **135编辑器可用** — 标准contenteditable iframe，保留内联CSS
3. **图片必须base64** — 否则粘贴后图片丢失
4. **内嵌版HTML是最终交付物** — 打开→全选→复制→135粘贴
5. **主题自动匹配** — wechat-publisher的html_converter根据话题类型自动选主题
